# Fraude en tarjetas de credito.

> **_OBJETIVO:_** Este ejercicio es para probar las bases de el machine learning y la ciencia de datos.<br><br>**_PLANTEAMIENTO:_** _Se utilizara un dataset recuperado de Kaggle_ el cual _recopila información sobre transacciones bancarias_, estas transacciones muestran los movimientos _anonimanizados_ de europeos, son _datos reales_ y mi _objetivo es demostrar como funcionan los modelos más basicos del machine learning_ en estas circunstancias.<br><br>Es decir estamos hablando de personas y no solo números, _las bases_ juegan un papel fundamental desde mi punto de vista y muchas ocasiones son tomadas a la ligera.Así que comenecemos planteando los problemas que persivo.

### Problemas.

- Bloqueos inesperados.
- Reactivación lenta.
- Generan frustración.
- Disminuye la confianza.

### METODOS.

> Para logar entender como funcionan las alertas de notificación de fraudes usaremos la teoría de conjuntos, la matriz de cofusión, el teorema de bayes y para medir su eficiencia lo haremos mediante Recall, Presicion, Medición de falsos positivos.
